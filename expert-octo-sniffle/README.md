# expert-octo-sniffle

Welcome to me simple Web-App that list a few climbing routes nearby to Paris.
 - API: https://www.mountainproject.com/data
 - Vuex
 - Vue-Router
 - Axios


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Run unit tests
```
vue-cli-service test:unit
```
